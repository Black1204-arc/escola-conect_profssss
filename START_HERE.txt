⚡ COMANDOS ÚTEIS
───────────────────────────────────────────────────────────────────────────────

INICIAR SISTEMA
  docker compose -f docker-compose.v2.yml up -d --build

VER LOGS
  docker compose -f docker-compose.v2.yml logs -f backend-v2
  docker compose -f docker-compose.v2.yml logs -f frontend-v2

PARAR SISTEMA
  docker compose -f docker-compose.v2.yml down

RESETAR (apagar dados)
  docker compose -f docker-compose.v2.yml down -v

ACESSAR SHELL DO BACKEND
  docker compose -f docker-compose.v2.yml exec backend-v2 sh

RODAR TESTES
  node test-v2.js

     (todos devem estar "healthy")

PORTA EM USO?
  → Mudar em docker-compose.v2.yml
  → ou: lsof -i :3000 && kill -9 <PID>

API NÃO RESPONDE?
  → docker compose -f docker-compose.v2.yml logs backend-v2
  → Aguarde migrations (~20s)

RESETAR BANCO?
