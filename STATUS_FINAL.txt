🚀 COMO USAR:

OPÇÃO 1 - Hot-reload (Desenvolvimento):
$ docker-compose watch

OPÇÃO 2 - Background (Produção-like):
$ docker-compose up -d

OPÇÃO 3 - Com Make (Mais fácil):
$ make dev          # Hot-reload
$ make up           # Background
$ make logs         # Ver logs
$ make clean        # Remover tudo

🛠️ TROUBLESHOOTING:

Problema: "Página em branco"
Solução: docker-compose logs frontend | tail -50

Problema: "Não consigo adicionar usuário"
Solução: docker-compose logs backend | tail -50

Problema: "Dados não persistem"
Solução: Verificar se MongoDB está rodando (docker-compose ps)

Problema: "Hot-reload não funciona"
Solução: docker-compose down -v && docker-compose watch
