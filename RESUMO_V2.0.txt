## 🐛 TROUBLESHOOTING

### Porta em uso?
```bash
# Mudar em docker-compose.v2.yml
# ou kill o processo:
lsof -i :3000    # Find
kill -9 <PID>    # Kill
```

### Banco de dados não conecta?
```bash
docker compose -f docker-compose.v2.yml logs postgres
docker compose -f docker-compose.v2.yml down -v  # Reset completo
docker compose -f docker-compose.v2.yml up -d    # Start fresh
```

### API retorna 502?
```bash
docker compose -f docker-compose.v2.yml logs backend-v2
# Aguarde ~20 segundos para migrations
```
